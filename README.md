# Paralives Cheats

A cheats, tips, and guides website for [Paralives](https://paralives-cheats.com/), built with [Astro](https://astro.build/) and [Tailwind CSS](https://tailwindcss.com/).

**Live site:** [https://paralives-cheats.com/](https://paralives-cheats.com/)

## Pages

- **Home** - Overview and latest content
- **Gameplay Guides** - In-depth guides for Paralives gameplay
- **Building Tips** - Tips and tricks for building and design
- **Contact** - Get in touch
- **Privacy Policy / Terms / Cookie Policy / DMCA** - Legal pages

## Getting Started

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Tech Stack

- [Astro](https://astro.build/) - Static site generator
- [Tailwind CSS v4](https://tailwindcss.com/) - Utility-first CSS framework
- [@astrojs/sitemap](https://docs.astro.build/en/guides/integrations-guide/sitemap/) - Sitemap generation

## Deploy

```bash
./deploy.sh
```
